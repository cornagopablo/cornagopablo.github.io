Funciones Hash

Una función criptográfica Hash usualmente conocida como "función resumen", es un algoritmo matemático que transforma cualquier bloque arbitrario de datos en una nueva serie de caracteres con una longitud fija.

Independientemente de la longitud de los datos de entrada, el valor hash de salida tendrá siempre la misma longitud.

Normalmente, esta longitud es de 40 caracteres, pero podrá ser diferente dependiendo de la función elegida.

Por ejemplo, los 256 bits del SHA-256, nos entrega un código alfanumérico de 64 caracteres.


Las principales características de las funciones Hash son:

  • Cada conjunto de datos al que se le aplica la función nos dará un hash único, por muy pequeña que sea la variación en los datos (es muy difícil encontrar dos documentos que tengan el mismo resultado hash).

  • Otra de ellas es la prácticamente imposibilidad de obtener el conjunto de datos a partir del código alfanumérico. Es decir, es una función de un solo sentido.

Una página web que nos permite transformar texto plano a texto cifrado es Dan's Tools.


Son muy usados para proteger la confidencialidad de contraseñas, por ejemplo, para comprobar si una contraseña está en una base de datos. Se aplica la función Hash a la contraseña introducida y se compara con el resultado de aplicar una función Hash que será lo que está almacenado en la base de datos.

También se utilizan para garantizar la integridad de la información, por ejemplo, en la descarga de archivos grandes.
